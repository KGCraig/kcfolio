About Me

Hello! I'm Kameron Craig, a junior Data Analyst and Data Processing Specialist with a background in front-end webpage design, scripting and app development.

Organizing, analyzing and finding insights in data is something I really enjoy doing. I have fun creating and cleaning databases that can be used for machine learning and data analysis. I often use tools like Python, PostgreSQL and Google Sheets to clean and create databases, but I've also implemented tools like SQLite3 + DB Browser, Microsoft Power BI and Tableau.

I love to support the open-source scene, so I like contributing to and using open-source tools like Python, PostgreSQL and DB Browser.

I want to become a data scientist and machine learning engineer in the future with a focus on computer vision. I've always been interested in robots and how they interact with the world!

I've always enjoyed modifying and hacking video games as a hobby, and I've really improved my programming skills by doing such. I have a lot of passion for video games, so I would love to help in that area if you've got a project going on!

The projects on my website mostly lead to GitHub pages where you can see the exact structure of my projects. I value integrity and transparency, so I want people to see exactly how my projects were done.
